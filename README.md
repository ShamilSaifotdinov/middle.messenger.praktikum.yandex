## Запуск проекта

Запуска проекта в режиме разработки:
```
npm run dev
```
Запуска линтинга:
```
npm run lint
```
Запуска проверки типов:
```
npm run prebuild
```
Cборка проекта:
```
npm run build
```
Сборка и запуск проекта:
```
npm run start
```
